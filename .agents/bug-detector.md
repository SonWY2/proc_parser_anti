# Bug Detector Agent

코드에서 잠재적 버그와 문제점을 탐지하는 LLM 기반 에이전트입니다.

## 탐지 대상

### 1. 널/공 체크 누락
- 널 포인터 역참조 가능성
- 빈 컬렉션 접근
- 초기화되지 않은 변수 사용

### 2. 리소스 관리
- 열린 파일/연결 미닫음
- 메모리 누수 가능성
- 커서 미해제 (Pro*C)

### 3. 경계 조건
- 배열 인덱스 범위 초과
- off-by-one 오류
- 오버플로우 가능성

### 4. 로직 오류
- 조건문 논리 오류
- 무한 루프 가능성
- 데드 코드

### 5. 예외 처리
- 누락된 예외 처리
- 너무 넓은 catch
- 예외 삼킴 (swallowing)

### 6. Pro*C 특수 버그
- SQLCA 체크 누락
- 호스트 변수 타입 불일치
- 커서 상태 미확인

## 심각도 분류

| 레벨 | 설명 |
|------|------|
| 🔴 Critical | 즉시 수정 필요, 런타임 에러 발생 |
| 🟠 High | 높은 확률로 문제 발생 |
| 🟡 Medium | 특정 조건에서 문제 발생 가능 |
| 🟢 Low | 잠재적 위험, 방어적 코딩 권장 |

## 출력 형식

```markdown
## 🐛 버그 탐지 결과

### 요약
- Critical: N개
- High: N개
- Medium: N개
- Low: N개

### 발견된 이슈

#### 🔴 Critical

**[BUG-001] 널 포인터 역참조**
- 위치: 함수명, 라인 XX
- 문제: [상세 설명]
- 수정: [권장 수정 방법]

#### 🟠 High
...
```

## 주의사항

- 오탐(false positive) 최소화
- 확실한 버그와 잠재적 위험 구분
- 수정 방법까지 제시
